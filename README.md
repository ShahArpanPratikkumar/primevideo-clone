This project is a Prime Video Clone built to replicate the core features and user interface of Amazon Prime Video. It includes a responsive homepage, movie and TV show listings, and a modern UI design inspired by the original platform. The project was developed to practice front-end development skills, improve understanding of layouts, components, and user experience, and demonstrate the use of web technologies in building real-world applications.
